Caracteres Especiais

Exemplos de entidades HTML:

  • Espaço não separável: &nbsp; =>  teste
  • Sinal menor: &lt; => <
  • Sinal maior: &gt; => >
  • E comercial: &amp; => &
  • Copyright: &copy; => ©
  • Registrado: &reg; => ®
  • Euro: &euro; => €
  • Aspas duplas: &quot; => "
  • Aspas simples: &#39; => '
