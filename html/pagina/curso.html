Sobre o Curso

Este curso foi desenvolvido para pessoas que desejam iniciar ou evoluir na área de desenvolvimento web. Não é necessário conhecimento prévio, pois o conteúdo foi organizado de forma progressiva, clara e totalmente prática.

  • Introdução à Web
  • Como funciona o HTML
  • Tags essenciais

Módulo 2 – Estilização

  • Introdução ao CSS
  • Cores, fontes e espaçamentos
  • Layouts básicos

Módulo 3 – Lógica com JavaScript

  • Variáveis e operadores
  • Funções e eventos
  • Integração com HTML

Instrutor
